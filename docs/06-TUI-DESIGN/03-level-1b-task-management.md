# Level 1-B: 任务管理界面

> **配置查询任务的核心界面**

---

## 界面功能

1. 选择AI平台（多选）
2. 输入问题（单行或导入CSV）
3. 选择执行模式（串行/方案A/方案B/自动）
4. 选择输出模式（演示模式/写库模式）⭐

---

## 交互流程

### Tab切换焦点
- Tab键在4个步骤区域间切换
- 当前焦点区域高亮边框

### 步骤1: 选择AI
- Space键勾选/取消
- 未登录的AI显示灰色，无法选择

### 步骤4: 输出模式（关键）
- 演示模式 → Level 2-A → Level 3-A
- 写库模式 → Level 2-B → Level 3-B
