# Level 2-A: 执行中界面（演示模式）

> **实时显示查询进度，不写入数据库**

---

## 界面功能

1. 显示AI×问题进度矩阵
2. 实时日志滚动
3. 显示执行时间和剩余时间
4. 自动跳转到Level 3-A

## 交互流程

### 自动执行
- 无需用户操作，自动执行所有查询
- 完成后自动跳转到Level 3-A

### 中断处理
- 用户按Ctrl+C → 显示确认对话框
- 确认后：保存已完成的结果，跳转到Level 3-A
- 取消：继续执行
