# Level 3-A: 结果展示界面（演示模式）

> **逐个展示AI回复，支持对比和手动保存**

---

## 界面功能

1. 逐个展示AI回复（← →切换）
2. 并排对比视图（V键切换）
3. 手动保存（S键）
4. 导出功能（E键）

## 交互流程

### 按键响应
- **← →**：切换AI回复
- **V**：切换并排对比视图
- **S**：保存当前结果到数据库
- **C**：复制全部结果到剪贴板
- **E**：导出为CSV/JSON
- **Q**：返回主界面
