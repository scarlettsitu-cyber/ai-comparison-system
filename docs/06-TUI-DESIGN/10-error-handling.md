# 错误处理设计

> **统一的错误处理和展示策略**

---

## 错误类型

### 1. 登录错误
**场景**：
- 验证码过期
- 网络超时
- 账号被封禁

**处理**：
- 显示错误弹窗（boxen）
- 提示用户重试或切换AI
- 3秒后自动关闭

### 2. 查询错误
**场景**：
- AI响应超时
- 网络中断
- 问题格式不支持

**处理**：
- 写库模式：记录到失败任务列表
- 演示模式：显示错误标识，继续其他AI

### 3. 系统错误
**场景**：
- 内存不足
- 磁盘空间不足
- 数据库锁定

**处理**：
- 显示严重错误弹窗
- 建议用户清理资源或切换模式
- 按任意键返回主界面
