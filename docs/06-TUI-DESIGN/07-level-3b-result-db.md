# Level 3-B: 完成总结界面（写库模式）

> **显示完成统计和失败任务处理**

---

## 界面功能

1. 显示完成统计（成功/失败/总耗时）
2. 显示失败任务列表
3. 重试失败任务（R键全部重试、Enter单个重试）
4. 查看详情

## 交互流程

### 重试逻辑
- **R键**：重试所有失败任务 → 跳转到Level 2-B
- **Enter键**：重试选中任务 → 直接执行，完成后刷新列表

### 失败处理
- 重试3次后仍失败 → 标记为"永久失败"
- 用户可查看详细日志（D键）
